# Getting Started with Weather APp
Este proyecto fue iniciado y desarrollado con React, [https://axios-http.com/docs/intro](Axios) y [https://www.accuweather.com/](Accuweather)  

<details open="">
  <summary><h2>✨Acerca del proyecto</h2></summary>
  <br>
  <p>
    Una aplicación de clima es un software diseñado para proporcionar información sobre las condiciones meteorológicas actuales y pronósticos del tiempo. Estas aplicaciones recopilan datos de diversas fuentes, como estaciones meteorológicas, satélites, radares y modelos climáticos, y los presentan de una manera fácil de entender para los usuarios.
